Καθώς τα μέτρα προστασίας του COVID 19 θα επιβληθούν στα νηπιαγωγεία αλλά και σε όλες τις τάξεις του δημοτικού, δημιουργήσαμε μία ιστοσελίδα που περιλαμβάνει ένα παιχνίδι με εκπαιδευτικό χαρακτήρα. Τα παιδιά θα κληθούν να επιλέξουν σε ποιες περιπτώσεις τα μέτρα υγηίνεις εφαρμόζονται σωστά. Με τον τρόπο αυτό θα αναγνωρίσουν τα λάθη που μπορεί να κάνουν και θα τα εμπεδώσουμε. Είναι ένας έυκολοσ και διασκεδαστικός τρόπος που μπορεί να τους βοηθήσει να εφαρμόσουν σωστα τισ συμβουλές των ειδικών.


Σίγουρα θα υπάρχει ενθουσιασμός που θα ξαναβρεθούν οι φίλοι μεταξύ τους πόσο μάλλον εάν έχουν να συναντηθούν καιρό. Κάτι που προκαλεί άγχος είναι πλέον η σωματική επαφή και ειδικά στους γονείς πιο μικρών παιδιών. Ίσως περνούν σκέψεις όπως «Πώς θα πείσω το παιδί μου να μην αγκαλιάζεται, να μην φιλιέται με τα άλλα παιδιά»; Εδώ μπορείτε να ενισχύσετε τους «μυστικούς» χαιρετισμούς˙ δηλαδή να ανακαλύψουν οι φίλοι μεταξύ τους έναν χαιρετισμό που να μην προϋποθέτει την πολύ στενή επαφή. Ως απόρροια, θα αισθανθούν ότι ενισχύεται ο δεσμός τους μέσω του μυστικού τους κώδικα και θα προστατεύονται.
