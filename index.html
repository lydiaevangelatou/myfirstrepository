Καθώς τα μέτρα προστασίας του COVID 19 θα επιβληθούν στα νηπιαγωγεία αλλά και σε όλες τις τάξεις του δημοτικού, δημιουργήσαμε μία ιστοσελίδα που περιλαμβάνει ένα παιχνίδι με εκπαιδευτικό χαρακτήρα. Τα παιδιά θα κληθούν να επιλέξουν σε ποιες περιπτώσεις τα μέτρα υγηίνεις εφαρμόζονται σωστά. Με τον τρόπο αυτό θα αναγνωρίσουν τα λάθη που μπορεί να κάνουν και θα τα εμπεδώσουμε. Είναι ένας έυκολοσ και διασκεδαστικός τρόπος που μπορεί να τους βοηθήσει να εφαρμόσουν σωστα τισ συμβουλές των ειδικών.
